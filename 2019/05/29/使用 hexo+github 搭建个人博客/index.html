使用 hexo+github 搭建个人博客

1.电脑本地环境配置

  • 1.1 安装 git 和 node.js, 直接下载安装包，安装完成后 CMD 中输入node -v 和git --version 查看版本信息以确认是否安装成功。

  • 1.2 安装 HEXO并初始化 ，在新建的博客文件夹内右建打开git bash here 输入安装指令:npm i -g hexo， 初始化指令：hexo init。

2.网络端配置

2.1 GITHUB配置

  • 2.1.1 创建 github.io 项目： 注意项目名称与用户名称需一致, 如aidenhosea/aidenhosea.github.io 。

  • 2.1.2 添加 ssh key， 以能将本地文件通过 git 上传到 github 中:

    • 2.1.2.1 git bash 命令行下输入ssh-keygen -t rsa -C ‘你的邮箱地址’，然后回车，回车，再回车，一共3次回车键。
    • 2.1.2.2 生成密钥后，在你电脑目录：C:\Users\你的计算机用户名.ssh 下，会生成两个文件：私钥：id_rsa 和公钥：id_rsa.pub。
    • 2.1.2.3 输入cat id_rsa.pub 以查看并复制内容粘贴到https://github.com/settings/ssh ，点击添加新秘钥(New SSH key)，其中 Title 自己随便取，Key 就是把刚刚复制的都粘贴进来。

  • 2.2.3 重新部署上传博客。

3.优化 hexo 中的 next 主题

  • 3.1 下载： 从 https://github.com/iissnan/hexo-theme-next themes/next 下载并解压到博客中的 themes 文件夹。

  • 3.2 启用主题：打开站点配置文件中_config.yml配置文件 ，找到 theme 字段，并将其值更改为 next。如theme:next 。

  • 3.3 选择主题外观：本主题提供四种外观，搜索 scheme 关键字，将前面#去掉即生效 。

  • 3.4 设置语言：编辑 站点配置文件，language: zh-Hans 。

  • 3.5 设置头像：放置在 source/images/ 目录下并配置为：avatar: /images/avatar.png 。

  • 3.6 站点描述：编辑 站点配置文件， 设置 description 字段为你的站点描述。站点描述可以是你喜欢的一句签名:)。

  • 3.7 设置昵称：编辑 站点配置文件， 设置 author 为你的昵称。

    • 标签：博客目录 git bash here ,输入hexo new page tags ；
